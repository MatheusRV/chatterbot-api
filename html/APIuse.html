Go to:

https://localhost:8900/getResponse/{{ API-Token }}/{{ message }}

API-Token: Your API-Key that you can generate here.

message: The message to generate response for.

It will result in a JSON object with your response.

{
_id : ID_OF_USER,
username : YOUR_USER_NAME,
apitoken : YOUR_API_TOKEN,
query : YOUR_QUERY_OR_MESSAGE,
response : RESPOSE_FOR_QUERY
}

This JSON Object can then be used for expanding ChatterBot's usage among different platforms.
